# Tech Stack

## Framework & Runtime

- Framework: Next.js (App Router)
- Language: TypeScript
- Runtime: Node.js

## Core Dependencies

- Animation: Framer Motion
- Icons: Lucide React
- State: React Context / localStorage
- CSS: Vanilla CSS or Tailwind (project-specific)

## Build & Deploy

- Build: `npm run build`
- Dev: `npm run dev`
- Deploy: Vercel / Cloudflare Pages / Static server
